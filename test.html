Let's find the correct Oracle service names on your system.

Step 1: Run this command in Command Prompt as Administrator

sc query | findstr Oracle

Step 2: Or this command to get more details

net start | findstr Oracle

üéØ Manual Service Start (Quick Fix)

Based on your service name, try starting these services manually:

Option 1: Via Services.msc

  1. Press Windows + R, type services.msc, press Ctrl + Shift + Enter
  2. Look for services starting with "Oracle"
  3. Start these services (right-click ‚Üí Start):
    • OracleServiceXE (or similar with "Service" in the name)
    • OracleOraDB21Home1TNSListener (the one you found)

Option 2: Via Command Prompt as Administrator

net start "OracleServiceXE" net start "OracleOraDB21Home1TNSListener"

Note: The exact service names might be slightly different. Look for:

  • A service with "Service" in the name (this is the main database service)
  • A service with "TNSListener" in the name (this is the network listener)
